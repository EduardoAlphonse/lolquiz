https://lolquiz.eduardoalphonse.vercel.app

## Como executar o projeto
```bash
# clonar o repositório
git clone https://github.com/EduardoAlphonse/lolquiz.git

# entrar no diretório
cd lolquiz

# instalar as dependências
yarn
# ou
npm install

# iniciar servidor
yarn dev
# ou
npm run dev
```
